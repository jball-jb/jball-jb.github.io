Como Funciona HTML

HTML es un codigo que funciona de cierta forma como una jerarquia

Estructura del codigo HTML

index.html 

      <!DOCTYPE html> -- Es una declaracion que le dice al navegador que estamos programando con HTML 5
      <html lang="es"> -- Es la etiqueta raiz que envuelve todo el contenido de la pagina lang="es" especifica que el idioma original como español
        <head> -- Es la etiqueta que contiene metadatos y el titulo
          <meta charset="UTF-8"> -- Esta etiqueta define los metadatos
          <title>Mi página</title> -- Es la etiqueta que contiene metadatos
        </head> -- La etiqueta con </head> es la que cierra la etiqueta
        
        <body> -- Es la etiqueta que contiene metadatos
          <h1>Hola</h1> -- Es la etiqueta que contiene metadatos
        </body> -- Es la etiqueta que contiene metadatos
      </html> -- Es la etiqueta que contiene metadatos
