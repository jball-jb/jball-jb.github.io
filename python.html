¿Qué es Python?

Es un lenguaje de programación de alto nivel, interpretado y con una sintaxis muy limpia, lo que lo hace ideal para principiantes.

  • Legibilidad: Parecido al inglés humano.
  • Versatilidad: IA, Web, Ciencia de Datos.
  • Gran Comunidad: Miles de librerías gratuitas.

Tu primer código

print("¡Hola, mundo!")

Con solo una línea de código puedes empezar a interactuar con el sistema.
